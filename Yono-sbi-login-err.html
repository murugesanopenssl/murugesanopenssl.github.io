If mobile using Wi-Fi internet at home,
and mobile app using Yono sbi report following error:
	We are observing some technical issue. Please try again after some time.
Perform following actions inside mobile when using Wi-Fi at home:
Complete following actions:
1)
Open Settings
Click Connections
Enable and click Wi-Fi
Assume that you are having two Wi-Fi names:
	a) murugesan_openssl
	b) murugesanopenssl-5G
Initially connect to murugesan_openssl
At right side of connected murugesan_openssl network click Settings
Click View more
Scroll down and click MAC address type:
If you are having Randomised MAC change it to Phone MAC
Complete following actions:
2)
Open Settings
Click Connections
Enable and click Wi-Fi
Assume that you are having two Wi-Fi names:
	a) murugesan_openssl
	b) murugesanopenssl-5G
Now connect to murugesanopenssl-5G
At right side of connected murugesanopenssl-5G network click Settings
Click View more
Scroll down and click MAC address type:
If you are having Randomised MAC change it to Phone MAC
3)
Open Play store
Click search
Search the following:
	YONO SBI
Check if any updates available for YONO SBI
if available install the update.
4)
Disable Wi-Fi and enable Wi-Fi
5) After this we can login at YONO SBI mobile app.
If these steps are not working perform the same steps and use Randomised MAC instea of using Phone MAC
