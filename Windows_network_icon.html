It sounds like the "Turn system icons on or off" section in Settings has become outdated, and you're unable to toggle the network icon.

This is likely due to a bug or a change made by Microsoft. However, there's still a way to enable or disable the network icon:

**Method 1: Use the Registry Editor**

1. Press the Windows key + R to open the Run dialog box.
2. Type `regedit` and press Enter to open the Registry Editor.
3. Navigate to the following registry key:
        * HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\CurrentVersion\Explorer
4. Find the key called "NetworkIconState" and double-click on it.
5. If the value is not present, create a new DWORD (32-bit) value named `NetworkIconState` by right-clicking and selecting "New"
> "DWORD (32-bit) Value".
6. Set the value to 1 to enable the network icon or 0 to disable it.

**Method 2: Use a third-party utility**

There are some third-party utilities available that can modify system settings, including enabling or disabling the network
icon. You can search for these utilities online and follow their installation instructions.

Some examples of utilities that may allow you to enable or disable the network icon include:

* CCleaner
* Network Icon Tweaker
* System Configuration (sfc)

Please be cautious when using third-party utilities, as they may have unintended consequences on your system.

**Workaround: Use the "Network Icon" feature in Taskbar Settings**

If you're unable to modify the network icon through Settings or Registry Editor, you can try using the "Network Icon" feature in
Taskbar Settings:

1. Press the Windows key + I to open Settings.
2. Click on "System".
3. Click on "Taskbar".
4. Toggle off the switch next to "Network icon hidden".
