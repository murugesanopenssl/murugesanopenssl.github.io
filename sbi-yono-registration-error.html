STEPS TO RESOLVE YONO SBI REGISTRATION ISSUE:
  1. Ensure BSNL SIM is active and can send SMS.
  2. Check mobile network signal and restart phone.
  3. Install YONO SBI app from Play Store.
  4. Open app and select "Existing User".
  5. Enter username and password if you knew internet banking or Enter last 6 digits of your ATM card.
  6. Allow SMS and phone permissions.
  7. Complete SMS verification using registered SIM.
  8. Create NEW MPIN (do not reuse last 3 MPINs).
  9. Complete registration successfully.
